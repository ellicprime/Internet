This page contains infromation on Hypertext Transfer Protocol and the Domain Name System



HTTP and DNS make up a large portion of how we browse the web. HTTP is what makes it possible for you to load websites and interact with them. It's the set of rules that governs how your browser communicates with websites. DNS, on the other hand, is like the internet’s address book. Instead of typing out a IP adress, you can just type in a website’s name, and DNS handles finding where that site lives. Together, they make the internet easy to use.
